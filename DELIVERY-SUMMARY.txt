  Honeypot:      Cowrie SSH honeypot
  
  Default:       3 replicas (scalable 1-100+)
  Memory:        256Mi request, 512Mi limit per pod
  CPU:           100m request, 500m limit per pod

📝 COMMANDS TO REMEMBER

  # See all pods
  kubectl get pods -l app=cowrie -o wide
  
  # See logs
  kubectl logs -l app=cowrie --tail=50
  
  # Connect via SSH
  ssh -p 30222 root@<node-ip>
  
  # Scale to 5 pods
  kubectl scale deployment cowrie-honeypot --replicas=5
  
  # Delete and auto-recreate pod
  kubectl delete pod <pod-name>
